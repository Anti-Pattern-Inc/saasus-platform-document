以下の機能についてリリースを行いました。

メールによる多要素認証機能の追加  
多要素認証の方式として、従来の認証アプリに加え、メールによる認証コード送信に対応しました。  
ユーザーごとに認証アプリまたはメールのいずれかを多要素認証の方式として設定できます。  

　主な機能
  - メールによる多要素認証
    ログイン時に、登録済みのメールアドレスへ認証コードを送信し、二要素目の認証を行います。  
  - ユーザーごとの認証方式選択
    多要素認証の方式として、認証アプリまたはメールのいずれか一方をユーザーごとに設定できます。  
    両方を同時に設定することはできません。  

　ご利用条件
  - 独自ドメインによる認証メールの配信が有効になっているアカウントでのみご利用いただけます。
    https://settings.console.saasus.io/basic-info
