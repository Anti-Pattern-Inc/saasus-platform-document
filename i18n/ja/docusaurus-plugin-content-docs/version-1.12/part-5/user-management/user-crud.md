## ユーザーの検索

ユーザーは以下の条件で絞り込んで検索できます。

**検索可能な項目：**

- **ID・メールアドレス・ログインID・テナント名**：テキスト入力
- **テナント**：テキストボックス（オートコンプリート付き）
- **環境**：セレクトボックス
- **役割**：セレクトボックス
- **テナントに属していないユーザーも表示する**：チェックボックス

![検索画面](/ja/img/part-5/user-management/user-crud/user-search.png)

## ユーザーの新規作成

画面右上にある「新規ユーザーの作成」ボタンをクリックすると、ユーザー登録用のウィンドウが開きます。  
ユーザー登録用ウィンドウの上部で、以下の2つのタブを選択できます：


- メールアドレスで作成：メールアドレス認証でユーザーを作成する
- ログインIDで作成：ログインID認証でユーザーを作成する

選択したタブに応じて入力項目が切り替わり、
「メールアドレス」または「ログインID」のどちらか一方のみが表示されます。

:::warning 重要
「ログインIDで作成」で作成したユーザーをログイン可能にするには、
**認証詳細設定 > 基本設定 > ログインID** の
**「ログインIDの表記を追加する」** を ON にしておく必要があります。
OFF の場合、ID認証ユーザーを作成してもログイン画面でログインIDを使ってログインできません。
:::

ここでは入力項目について説明します。

**入力項目：**

| 項目名         | 説明                                                                 |
|----------------|----------------------------------------------------------------------|
| 作成方法       | 「メールアドレスで作成」または「ログインIDで作成」タブを選択します |
| メールアドレス | 「メールアドレスで作成」タブを選択した場合に表示される入力項目です。ユーザーのログインに使用されます |
| ログインID     | 「ログインIDで作成」タブを選択した場合に表示される入力項目です（半角英数字、ハイフン（-）、アンダースコア（_）が使用可能、最大50文字） |
| パスワード（任意） | ログイン用のパスワードを入力します。未入力の場合は一時パスワードが発行されます<br/>メールアドレスで作成する際、パスワードが未入力であれば、そのアドレス宛に一時パスワードが送信されます |
| テナント       | ユーザーが所属するテナントを選択します                              |
| ユーザー属性   | あらかじめ定義されたユーザー属性（例：ユーザー名、部署、役職など）を入力します |

![作成フォーム](/ja/img/part-5/user-management/user-crud/user-create.png)


## 既存ユーザーのテナントへの追加

「新規ユーザーの作成」ボタンのセレクトボックスから  
**「既存ユーザーのテナントへの追加」** を選択すると、追加用のウィンドウが表示されます。

この機能を使うことで、既に登録済みのユーザーに新たなテナントを割り当てることができます。  

**入力項目：**

| 項目名         | 説明                                                       |
|----------------|------------------------------------------------------------|
| メールアドレス / ログインID | 追加対象の既存ユーザーのメールアドレス、またはログインIDを入力します |
| テナント       | 割り当てるテナントを選択します                            |
| ユーザー属性   | あらかじめ定義されたユーザー属性（例：ユーザー名など）を入力します |

![テナント追加画面](/ja/img/part-5/user-management/user-crud/add-tenant-to-user.png)

## ユーザーの編集・削除

ユーザー一覧の各行にある「編集」または「削除」アイコンをクリックして操作します。

### 編集

編集アイコンをクリックすると、ユーザー情報の編集ウィンドウが表示されます。  
以下の内容を更新できます。

**編集可能な項目：**

- メールアドレス（認証方式がメールアドレスの場合）
- ログインID（認証方式がログインIDの場合）
- パスワード（再設定）  
- ユーザー属性（例：ユーザー名、部署など）

**実行可能な操作：**

- パスワードリセット（現在のパスワードを無効化し、新しいパスワードを設定）
- MFA解除（MFAを設定しているユーザーのみ、画面最下部にボタンが表示されます）

![編集画面](/ja/img/part-5/user-management/user-crud/user-edit.png)

### 削除

削除アイコンをクリックすると、**削除用のウィンドウ**が表示され、削除方法を選択できます。

**選択肢：**

- **テナントから削除**  
　ユーザーが選択されたテナントから削除されます。他のテナントには影響しません。

- **SaaSから削除**  
　ユーザー自体が SaaSus Platform から完全に削除され、全テナントから除外されます。

![削除ウィンドウ](/ja/img/part-5/user-management/user-crud/user-delete.png)

**注意事項：**

- 「SaaSから削除」を選択した場合、他のテナントに所属していても一括削除されます。
