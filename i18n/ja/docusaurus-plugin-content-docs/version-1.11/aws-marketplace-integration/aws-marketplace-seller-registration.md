### 事前準備

AWS Marketplace用にAWSアカウントを用意してください。

### 全体の流れ

出品手続きを始めるにあたって、まずお客様自身で以下の手続きを実施していただく必要があります。  
AWS Marketplaceへの出品に必要な登録、設定となるため忘れずに実施してください。

1. セラー登録
2. Public profileの設定
3. 税務情報に関するインタビュー
4. DAC7の税務アンケート
5. 銀行口座の設定
6. 支払い方法の設定
