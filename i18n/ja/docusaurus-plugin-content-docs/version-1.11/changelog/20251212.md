以下の機能についてリリースを行いました。

- ダッシュボード機能（Usage Metrics）  
  サービスの利用状況や成長推移を可視化する「ダッシュボード」機能を追加しました。  
  SaaSus Platformの認証・認可機能をご利用いただくだけで、蓄積されたデータをもとに自動的にメトリクスが集計・グラフ化されます。  
  これにより、お客様は追加の実装や特別な設定を行うことなく、日々のユーザー増加やテナント獲得の推移をビジネスの成長指標として直感的に把握できるようになります。

  主な機能

  - テナント数の推移表示  
    日付ごとに作成されたテナント数の推移をグラフで表示します。  
    B2B SaaSにおける導入企業数の増加トレンドを一目で確認することが可能です。  
  - ユーザー数の推移表示  
    日付ごとに作成されたユーザー数の推移をグラフで表示します。  
    サービスの規模拡大の状況や、登録ユーザーの成長曲線を可視化し、現状分析に役立てることができます。

  今回は第一弾として上記2つのメトリクスを公開しました。今後もビジネス判断に活用できる様々な指標を順次追加していく予定です。
