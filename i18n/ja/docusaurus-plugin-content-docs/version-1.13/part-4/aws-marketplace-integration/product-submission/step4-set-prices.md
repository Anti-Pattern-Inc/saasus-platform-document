続いて、AWS Marketplace上で販売する製品の購入オプションと契約期間を入力していきます。

AWS Marketplace連携機能で出力された指示書に従って、`Purchasing options`と`Contract duration options`を入力してください。

:::caution

必ず指示書に従って入力してください。

[AWS Marketplace連携機能ページ](https://settings.console.saasus.io/integrations/aws-marketplace)から出品手続きを開始していただき、`ステップ1-4:SaaS製品情報の登録`まで進めていただくと指示書を確認できるようになります。

:::
