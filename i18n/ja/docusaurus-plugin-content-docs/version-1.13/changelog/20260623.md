以下の機能の提供停止についてお知らせいたします。

AWS Marketplace 連携機能の提供停止  
　2026年6月より AWS Marketplace のインテグレーション仕様が変更されました。これに伴い、現在の AWS Marketplace 連携機能（出品・掲載）は新しい仕様に未対応となり、従来の手順では出品（掲載）を完了できなくなっております。  
　つきましては、AWS Marketplace 連携機能の新規のご利用を停止させていただきます。  

　■ 影響
  - AWS Marketplace 連携機能を利用した新規の出品（掲載）を完了できません。

　■ 今後について
  - 仕様対応の目処が立ち次第、提供再開の時期について改めてご案内いたします。

　■ 最新の出品プロセスについて
  - 最新の出品プロセスや掲載手順については、新しいサービス解説サイト（[https://docs.deshima.cloud](https://docs.deshima.cloud)）をご参照ください。

ご不便をおかけしますが、何卒ご理解いただけますようお願い申し上げます。
