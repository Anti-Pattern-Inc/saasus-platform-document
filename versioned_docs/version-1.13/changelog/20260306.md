We have released updates on the following features.

Email-Based Multi-Factor Authentication  
In addition to authenticator apps, multi-factor authentication (MFA) now supports email-based verification codes.  
Users can choose either an authenticator app or email as their MFA method on a per-user basis.  

　Main features
  - Email-Based MFA
    Upon login, a verification code is sent to the user's registered email address for second-factor authentication.  
  - Per-User Authentication Method Selection
    Each user can configure either an authenticator app or email as their MFA method.  
    Both methods cannot be enabled simultaneously for a single user.  

　Requirements
  - This feature is available only for accounts with custom domain email delivery enabled.  
    https://settings.console.saasus.io/basic-info
