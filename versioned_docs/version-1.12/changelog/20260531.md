We have released updates on the following features.

New UI for User Management and Feature Improvements  
　We have launched a new UI (management screen) for the User Management screen, designed to improve usability and performance.  

　Main features
  - Performance Improvement (Faster Search)
    As the number of users grew, search operations had been taking longer to complete. With this release, we optimized internal processing and significantly improved performance.  
  - User ID Search
    In addition to the existing search conditions, you can now filter search results by "User ID".  
  - Integration of the Role Management Screen
    Along with the migration to the new UI, the conventional "Role Management" screen has been retired. The same management and configuration can now be performed within the new UI.  

　■ How to use the new UI  
　To minimize the impact on existing operations, the conventional UI is displayed by default. To try out the new UI, please follow the steps below to switch.  

  1. Open the User Management screen.
  2. Turn ON the "Try the new UI (toggle switch)" placed on the screen.

　※ You can return to the conventional UI at any time by turning the toggle switch OFF.  

With this release, user management can be performed more comfortably and smoothly even in environments with a large number of users. Please try out the comfortable operability of the new UI.
