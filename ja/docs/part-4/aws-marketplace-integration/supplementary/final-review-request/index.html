最終レビュー依頼の方法について

出品手順「10. AWS Marketplaceとの連携確認」が完了したら、AWS Marketplaceサポートチームへ最終レビューの依頼を行います。

まずAWS Marketplace Management Portalにアクセスしていただき、対象のプロダクト詳細ページを開きます。

次に画面上部に表示されているアラートの右側のUpdate visibilityを押下します。

遷移後にVisibility optionsをpublicに設定します。

追加で契約金額を入力するフォームが追加で表示されるので、正規の金額を入力した後Submitを押下します。

以上で最終レビュー依頼は完了です。
