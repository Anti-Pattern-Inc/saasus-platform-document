銀行口座の設定

続いてPayment InformationよりComplete banking informationを押下して銀行口座を設定します。
米国の銀行口座の登録が必要となり、無い場合は、Hyperwalletで仮想のUS bank account number
（Deposit Account）を取得し、指定の日本の銀行口座で受け取ることが可能となります。

米国の銀行口座についての質問に回答します。

米国の銀行口座を保持していない場合、Hyperwallet アカウントについて回答します。

Hyperwallet アカウントを保持していない場合、Sign up for Hyper walletからアカウントを作成します。

Hyperwalletの登録についてはこちらの手順を参照ください。

Hyperwalletの登録で取得した情報を米国銀行口座として登録します。

必要な情報を登録後Submitを押下し銀行口座の設定を行います。

以上で、銀行口座の設定が完了となります。
