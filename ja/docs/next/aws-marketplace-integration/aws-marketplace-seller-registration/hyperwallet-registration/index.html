Hyperwalletの登録

銀行口座の登録からSign up for Hyper walletを押下すると以下画面に遷移します。
アカウントの確認方法、有効コードを押下します。

有効コードの入力となるので、遷移前の画面のSign up for Hyper wallet下にあるPin Codeを入力して続けるボタンを押下してください。

入力後アカウントタイプの入力を求められるので環境に合わせて選択してください。

職種を選択すると、プロフィール情報の入力を求められるので、企業情報、会社住所等を入力してください。

内容を入力後は続けるを押下してください。

引き続き、会社連絡先情報、会社連絡先住所等を入力してください。

内容を入力後は続けるを押下してください。

引き続き、Eメールアドレスとパスワード、秘密の質問、法的契約等を入力してください。

内容を入力後は確認を押下してください。

確認するとアカウントが正常に有効化されましたと表示されます。
有効になったらAdd　Trunsfer Methodを押下します。

転送先の口座を指定してContinueを押下します。

遷移後転送先の銀行口座情報を入力後Submitを押下します。

Submit後米国の預金口座が作成されます。

登録後、銀行口座の設定に戻ります。
