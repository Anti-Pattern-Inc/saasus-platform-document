We would like to inform you about the suspension of the following feature.

Suspension of the AWS Marketplace Integration Feature  
　Due to changes to the AWS Marketplace integration specification effective June 2026, the current AWS Marketplace integration feature (product listing) no longer supports the new specification, and listing can no longer be completed with the existing procedure.  
　Accordingly, we have suspended new usage of the AWS Marketplace integration feature.  

　■ Impact
  - New product listings using the AWS Marketplace integration feature cannot be completed.

　■ Outlook
  - We will announce the timing of resumption once a plan to support the new specification is in place.

　■ About the latest listing process
  - For the latest listing process and procedures, please refer to our new service guide site ([https://docs.deshima.cloud](https://docs.deshima.cloud)).

We apologize for any inconvenience this may cause and appreciate your understanding.
